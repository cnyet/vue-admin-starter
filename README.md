# vue-build
> 基于 @vue/cli 4.3.1 官方开发者工具配置的一套适合开发中后台系统的脚手架，包括HTTP请求的封装，模块分组，CSS样式设置，以及常用插件。

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Run your end-to-end tests
```
yarn test:e2e
```

### Lints and fixes files
```
yarn lint
```
